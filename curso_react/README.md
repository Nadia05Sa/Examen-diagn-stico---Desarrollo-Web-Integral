# Frontend — Examen diagnóstico (React + Vite)

## Nombre del proyecto
**Frontend — Examen diagnóstico — Desarrollo Web Integral**

## Descripción
Aplicación web en **React** creada con **Vite** para consumir la API del backend (Django REST Framework) y permitir la gestión/visualización de cursos.

## Tecnologías utilizadas
- **React**
- **Vite**
- **JavaScript**
- **npm**

## Funcionalidades
- **Consumo de API** de cursos (listar/crear/editar/eliminar, según implementación del backend y la UI)
- **Interfaz web** para trabajar con los datos del sistema

## Instrucciones para ejecutar el proyecto
1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar en modo desarrollo:

```bash
npm run dev
```

3. (Opcional) Build de producción:

```bash
npm run build
```

### Configuración de API
Revisa/ajusta la URL del backend en `src/service/api.js` para que apunte a tu servidor Django (por ejemplo `http://127.0.0.1:8000/`).
